# 🚗 DriveEase - Car Rental System

## About

DriveEase is a Full Stack Car Rental System developed using React.js, Spring Boot, and MySQL. The application allows users to browse available cars, book cars online, and manage bookings through a simple and responsive interface.

---

## Features

- View available cars
- Search cars
- Book a car online
- Booking confirmation
- Admin login
- Responsive design
- REST API integration
- MySQL database

## Installation

### Clone the repository

```bash
git clone https://github.com/gowthami-10-g/DriveEase.git
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
mvn spring-boot:run
```

---

## Database

Configure the MySQL database in:

```
backend/src/main/resources/application.properties
```

Update the database URL, username, and password before running the project.
